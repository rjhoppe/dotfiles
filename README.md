# Dotfiles

This repository contains my personal dotfiles and configurations for various tools and environments.

## Contents

- **editors**: Configurations for text editors (e.g., Neovim, VS Code)
- **productivity**: Scripts and configurations for productivity tools
- **shell**: Shell configurations (e.g., Bash, Zsh)
- **terminal**: Terminal emulator configurations
- **tmux**: Tmux configurations
- **window**: Window manager configurations
- **opencode**: OpenCode config files
